# Різьбовий Калькулятор — Thread Drill Size Calculator

PWA застосунок для визначення діаметру отвору (свердла) під нарізання різьби мітчиком.

## Функції

- 🔩 **Класичний мітчик / різьбофреза** — 62 розміри M1.4–M42
- ⬡ **Безстружковий мітчик (розкатник)** — M2–M12
- 🔍 **Пошук** за розміром різьби (M8, M10x1.5...)
- ⇄ **Режим порівняння** класичний vs розкатник
- 📱 **PWA** — встановлюється на домашній екран
- ✈️ **Офлайн режим** — працює без інтернету

## Деплой на Netlify

1. Форкни або завантаж репозиторій
2. Підключи до [Netlify](https://netlify.com)
3. Build command: *(порожньо)*
4. Publish directory: `.` (корінь)
